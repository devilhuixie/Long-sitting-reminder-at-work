# 久坐提醒

## 用于上班久坐提醒,预防减轻痔疮;为了健康
## 仅限Windows使用

目前只在Windows11下测试使用过.

会在每天的10点,11.30 13.30 15.00以及16.30提醒久坐.(因为18点下班了就不提醒了,跳过了午休一小时)
